# flyProject
## 基於無人機平台之多光譜偏振影像系統 (Multispectral Polarization Imaging System)
*Fun Fact: 本專案最初的開發宗旨為「針對張詠筑碩士能不能畢業開發的新專案」—— 並且非常成功地完成了使命！*

### 專案簡介 (About)
本專案開發了一套基於樹莓派 (Raspberry Pi) 控制、搭載自動旋轉色輪與液晶 FLIR 偏振相機的輕量化遙測系統。透過高度整合的軟硬體架構與 4G LTE 遠端圖傳技術，本系統能穩定掛載於無人機（如 DJI Agras T10）上，實現戶外複雜環境下的即時多光譜偏振影像擷取、遠端控制與地物辨識。

### 核心功能 (Features)
多光譜偏振擷取：整合自製 3D 列印旋轉色輪與步進馬達，可遠端切換多波段濾光片（可見光、紅、綠、藍光）進行拍攝。
全方位偏振參數運算：透過自主開發的影像處理程式，能讀取 .raw 與 .tiff 格式，拆解 0°、45°、90°、135° 偏振影像，並計算史托克參數 (Stokes S0, S1, S2)、線偏振度 (DoLP) 與偏振角 (AoP)。
跨平台行動控制介面：建構基於 Web (HTML + Python Flask) 的地面遙控軟體，支援手機瀏覽器直接觀看即時影像串流、控制馬達、觸發拍照。
高可靠雲端備份：具備自動化影像輸出機制，拍攝完畢後可自動將大量原始資料備份至 NAS 伺服器，確保高空作業資料不遺失。
### 硬體架構 (Hardware Architecture)
系統總重量僅 592 克，兼顧輕量化與穩定性：
核心控制板：Raspberry Pi 4B
偏振相機：Teledyne FLIR BFS-U3-51S5P-C (Sony IMX250MZR 全局快門 CMOS，支援像素級偏振濾光)
色輪驅動：微步驅動步進馬達 + AD4980 驅動晶片
通訊模組：SIM7600G-H (4G LTE 模組)
電源管理 (BMS)：客製化 18650 鋰電池升壓電路，提供系統穩定 5V 獨立供電與低電量安全關機保護
### 目錄結構 (Directory Structure)
專案主要使用 C 語言與 Python 開發，核心目錄說明如下：
 -  Drawing/：機構硬體設計檔，包含 3D 列印之外殼圖檔與旋轉色輪設計檔。
 -  stepmotor/：步進馬達控制程式，負責精準定位色輪與濾光片切換。
 -  PolarizerCamera/ &
 -  rawReader/：前端 Web 操作介面與後端伺服器架構，包含修改後的手機自適應版面與 NAS 雲端備份功能。
 -  read_tiffandraw/：核心影像與數據處理引擎，包含讀取相機原始偏振影像、修正偏振讀取錯誤，以及「解偏振」等高階運算程式。
 -  Datasheet/ & Reference/：硬體規格書與開發參考文件。
 -  應用場景 (Applications)

本系統已成功透過地面與戶外無人機飛行實驗驗證，能有效運用偏振特性 (DoLP/AoP) 區分出傳統可見光難以辨識的目標：
植被辨識與監測：精準分辨真實植物與紡織品偽裝。
材質分類：有效區分複雜場景中的金屬與塑膠製品。
水下與隱蔽物探測：成功偵測沙地上的軍用偽裝網與水面下的金屬鐵片。
